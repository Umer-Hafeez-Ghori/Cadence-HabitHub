 Cadence HabitHub - Minimalist Habit Tracker

Cadence HabitHub is a simple habit tracking web application designed to make daily habit building easier and more enjoyable. The project focuses on a clean user experience, smooth interactions, and straightforward code that is easy to understand and maintain.

 Key Features

• Dynamic Layout Toggling: The interface automatically switches between an empty-state view and the habit tracking dashboard whenever habits are added or updated.

• Positive Reinforcement Engine: Friendly success messages such as "Brilliant!" and "Star performance!" appear when habits are completed, providing instant feedback and encouragement.

• Integrated Quote Widget: Displays motivational quotes in the sidebar to make the interface feel more engaging and inspiring.

• Dynamic Streak Generation: Tracks and displays current habit streaks based on completed activities.

• State Continuity: Stores habit data in the browser using local storage, allowing progress to remain available between sessions.

 Tech Stack & Architecture

The application is built entirely with native web technologies and does not require any external libraries or frameworks.

• HTML5: Provides the structure and content of the application.

• CSS3 Variables & Animations: Used for styling, theme customization, and smooth visual effects.

• Vanilla JavaScript (ES6+): Handles habit management, streak calculations, event handling, and local data storage.

 Deployment Instructions

1. Clone or download the project files.

2. Make sure the project folder contains:

   * index.html
   * style.css
   * script.js

3. Open index.html in any modern web browser, or run it using a local server such as VS Code Live Server.

The application is ready to use immediately with no installation steps or additional dependencies required.
